Markdown(MD)的语法说明

# #这是 H1

## ##这是 H2

单行文本框

这是一个单行的文本框,只要1个Tab再输入文字即可

多行文本框

	这是一个有多行的文本框
	你可以写入代码等,每行文字只要输入2个Tab再输入文字即可
	这里你可以输入一段代码

比如我们可以在多行文本框里输入一段代码,来一个Java版本的HelloWorld吧

    public class HelloWorld {
      /**
      * @param args
	    */
	    public static void main(String[] args) {
		    System.out.println("HelloWorld!");

	    }

    }

链接

链接内容定义的形式为：

方括号（前面可以选择性地加上至多三个空格来缩进），里面输入链接文字
接着一个冒号加空格或直接括号
接着链接的网址
选择性地接着 title 内容，可以用单引号、双引号或是括弧包着

例如:

	[点击这里你可以链接到www.baidu.com](http://www.baidu.com)<br />
	[点击这里我你可以链接到我的博客](http://luzexi.com)<br />

效果:

  1. 点击这里你可以链接到www.baidu.com
  2. 点击这里我你可以链接到我的博客

显示图片

	[]前面加!就代表图片了，其他和普通的连接差不多

列表

无序列表使用星号、加号或是减号作为列表标记：
*   Jesse
*   Sharon
*   Anne

效果:

  • Jesse
  • Sharon
  • Anne

等同于：

+   Jesse
+   Sharon
+   Anne

效果:

  • Jesse
  • Sharon
  • Anne

等同于：

-   Jesse
-   Sharon
-   Anne

效果:

  • Jesse
  • Sharon
  • Anne

#####有序列表则使用数字接着一个英文句点

1.  Jesse
2.  Sharon
3.  Anne

效果:

  1. Jesse
  2. Sharon
  3. Anne

等同于:

1. Jesse
1. Sharon
1. Anne

效果:

  1. Jesse
  2. Sharon
  3. Anne

等同于:

4. Jesse
2. Sharon
8. Anne

效果:

  1. Jesse
  2. Sharon
  3. Anne

分割线

	* * *

	***

	*****

	- - -

	---------------------------------------

效果:

test1

test2 *** test3

testend

强调

	*single asterisks*

	_single underscores_

	**double asterisks**

	__double underscores__

效果:

single asterisks

single underscores

double asterisks

double underscores
